Durable memory lives here.

Use this file for:
- stable human preferences
- durable agent operating rules
- validated conclusions
- long-lived project truths

Suggested entry format:
- **Signal:** what was observed
- **Conclusion:** what the agent now infers
- **Action implication:** what should change in future behavior

- **Source:** where this came from
